# MissNet 项目总结（交接版）

## 项目定位
MissNet 是一个 Android 原生应用（Kotlin + Jetpack Compose + Material 3），当前主线目标是：

1. 保持视频播放、下载、导出主能力可用
2. 将 UI/UX 逐步重构为更接近 Google Drive App 的成熟产品感
3. 在重构过程中持续修复滚动、布局、状态表达等稳定性问题

## 当前已完成的主线工作

### 1. 下载 / 导出主线
- 已接入 FFmpeg 能力
- 直链视频导出可用
- HLS / m3u8 已有最小可用导出链路
- Downloads 页已对接真实状态
- 已修过导出删除风险问题

### 2. UI/UX 主线
- 已建立系统化蓝图、计划、进度追踪文档
- Home 已从展示页转向“任务恢复入口”方向
- Library / Downloads 已从普通功能页转向“任务与资产状态中心”方向
- 已做至少两轮 Home / Library 页面重构与样式收敛
- 已做一轮滚动问题修复，避免主 tab 页面再次出现上下滑动失效

### 3. 工程与交付
- 多个测试 APK 已成功打包并通过 Telegram 发送测试
- 当前交接目标是让本地 Codex CLI 能无缝接手后续工作

## 当前产品方向（必须延续）
- UI/UX 对标：Google Drive App
- 关键词：克制、系统感、可扫描、状态明确、操作可预测、长期使用不累
- 当前不优先：真正离线 HLS 导出完整版
